# Política de Segurança

## Como reportar vulnerabilidades

Para reportar uma vulnerabilidade, abra uma issue privada pela aba **Security** do repositório.

## Versões suportadas

Atualmente, apenas a branch `main` é suportada com correções de segurança.

## SLA de resposta

O prazo inicial de resposta para novos reportes é de até **5 dias úteis**.
